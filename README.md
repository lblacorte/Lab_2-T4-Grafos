# T4 - LAB 2 - GRAFOS | OPERAÇÕES | BUSCA EM LARGURA E PROFUNDIDADE

Todas as funcionalidades pedidas foram implementadas, com grafo (representado com listas de adjacência) e fila (para percorrer em largura):
- Criar grafo de acordo com o número de vértices pedido pelo usuário;
- Inserir aresta no grafo;
- Remover aresta do grafo;
- Imprimir o grafo;
- Percorrer o grafo a partir de um nó informado pelo usuário por largura, mostrando o percurso em pré-ordem e em pós-ordem;
- Percorrer o grafo a partir de um nó informado pelo usuário por profundidade, mostrando o percurso em pré-ordem e em pós-ordem.

Arquivos:
- grafosComListas.c e .h - funções básicas de operações com grafos usando listas de adjacência;
- fila.c e .h - funções básicas de operações com filas, utilizadas para busca em largura;
- principal.c - demais funções adequadas para o contexto do programa para tratamento de um grafo.
